# 🧙‍♂️ OML&CO - Лавка колдовских вещей

![OML&CO Logo](https://avatars.mds.yandex.net/i?id=373ea4d4ea894939ab131788f4261194_l-6323775-images-thumbs&n=13)

**OML&CO** — это полноценное веб-приложение интернет-магазина, специализирующегося на продаже шуточных колдовских товаров. Проект состоит из двух частей: бэкенд на Node.js/Express и фронтенд на React.

## 🎯 О проекте

**OML&CO** — это лавка колдовских вещей с 1666 года. Здесь можно найти:
- ✨ Зелья (от похмелья, правды, удачи)
- 🪄 Волшебные палочки
- 🧹 Летающие мётлы
- 📚 Книги заклинаний
- И другие магические артефакты с юмором

Проект создан в рамках учебных практических занятий №4 и №5 по дисциплине "Фронтенд и бэкенд разработка".

## ✨ Функциональность

### Основные возможности:
- ✅ **Просмотр товаров** — все товары отображаются в виде красивых карточек
- ✅ **Фильтрация** — поиск по названию и фильтрация по категориям
- ✅ **Добавление товаров** — создание новых колдовских предметов через модальное окно
- ✅ **Редактирование** — изменение информации о существующих товарах
- ✅ **Удаление** — удаление товаров из лавки
- ✅ **Интерактивная документация API** — тестирование всех эндпоинтов через Swagger UI

### Дополнительно:
- Адаптивный дизайн для мобильных устройств
- Рейтинг товаров звездочками
- Индикация наличия на складе (цветовая маркировка)
- Красивые градиенты и hover-эффекты

## 🛠 Технологический стек

### Бэкенд:
- **Node.js** — среда выполнения JavaScript
- **Express.js** — веб-фреймворк
- **nanoid** — генерация уникальных ID
- **cors** — обработка CORS-запросов
- **Swagger UI Express** — документация API
- **swagger-jsdoc** — генерация спецификации из JSDoc

### Фронтенд:
- **React** — библиотека для построения интерфейсов
- **Axios** — HTTP-клиент для запросов к API
- **SCSS** — стилизация компонентов
- **CSS Grid/Flexbox** — построение сетки карточек
